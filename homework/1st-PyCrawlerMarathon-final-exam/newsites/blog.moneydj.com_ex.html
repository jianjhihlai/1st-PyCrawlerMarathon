特斯拉召回至少 1.8 萬台 Model X，轉向器有失去動力的安全疑慮

電動車製造商特斯拉宣布召回 1.5 台 Model X 電動車，這款產品在設計上存在問題，可能會導致電動車在轉向的過程中失去助力，召回的方案適用於 2016 年生產的 Model X，在 2016 年 10 月中旬之後製造的 Model X 則沒有這個設計問題。

特斯拉公司透露，在受設計問題影響的 Model X 電動車中，已確認部分連接轉向器的螺栓有著過度腐蝕的問題，主要發生在天氣寒冷的地區，如果螺栓失效，轉向過程中的助力會消失，特別是在低速行駛的過程中需要更多的作用力來轉動方向盤，這增加了失事的風險。

由於但目前還沒有收到任何與動力轉向組件故障相關的事故報告，因此在召回範圍內的 Model X 進行維修之前，車主可繼續使用 Model X，維修部門將在確認零組件到貨後與車主聯繫，除了更換螺栓外，維修的過程中還會加入防腐蝕的密封劑，以解決在寒冷天氣下的行駛問題。

涉及召回的 Model X 為 2016 年 10 月之前生產，在美國、加拿大、中國等多個市場發售，在美國市場有 14,193 台 Model X，在加拿大市場有 843 台 Model X，在中國市場有 3,183 台 Model X 在召回的範圍內，車主可聯繫特斯拉的售後服務確認。

這不是特斯拉第一次召回存在設計問題的電動車，之前該公司曾大規模召回 Model S，更換其輔助轉向的馬達螺栓，共涉及 12.3 萬台 Model S。
